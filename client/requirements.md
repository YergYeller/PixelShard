## Packages
framer-motion | For smooth UI transitions and game overlay animations
howler | For game sound effects and music
react-use | For keyboard input handling and window size hooks

## Notes
Game logic will be handled via HTML5 Canvas in a React component
Authentication uses the pre-configured Replit Auth
Backend routes are defined in shared/routes.ts and must be matched exactly
Images will be drawn procedurally on canvas to match the "pixel art" style without assets
